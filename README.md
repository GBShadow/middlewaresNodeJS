# middlewaresNodeJS

## Sobre

Esta api node foi criada aparti das video aulas disponibilizada pela Rocketseat,a api possui uma CRUD onde existe alguns middlewares contando o numero de requisições e verificando a existencia de projetos com o id passado nos parâmetros, 
e bem simples mas me mostrou bem como utilizar os middlewares.

## Como Usar

Basta baixa a dependencia **express** ou rodar o comando **yarn**

## Considerações

Espero que o codigo possa ajudar alguem, caso alguem queira dar alguma dica ou sugestão agradeço.
